19. Перечислите основные свойства сервера NODE.JS
Поддерживает ассинхронность, однопоточность, ориентирован на события, в основании EventLoop, Chrome V8

20. Что такое npm?
Node Package Manager - менеджер пакетов, входящий в состав Node.js. Нужен для скачивания пакетов из облачного сервера npm, либо для загрузки пакетов на эти сервера.

21. Поясните назначение HTTP-заголовка Content-Type.
В HTTP-запросе или ответе описывает содержимое. Типы содержимого представленный с использованием типа интернет-носителя. Тип интернет-носителя (также известный как тип MIME) указывает тип данных, которые содержит файл.

22. Поясните назначение функции require.
Подключение одного независимого модуля (файла) к другому

23. Поясните понятие «Модуль Node.js».
Модуль - набор функций и объектов (кусок кода, по факту) которые могут использоваться в других приложениях

24. Поясните понятие «Node.js built-in modules» («Node.js Core modules»). 
Node.js built-in modules - встроенные модули, которые можно использовать без докачивания
